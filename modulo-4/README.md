# Módulo 4 - Consultas SQL de negocio

Esta carpeta contiene la pre-entrega correspondiente al **Módulo 4** del curso de **Data Analytics de Coderhouse**.

Durante este módulo se desarrollaron consultas SQL orientadas a responder preguntas de negocio utilizando la base de datos **Ventas_Tech_DB**, creada en el módulo anterior.

---

## Contenido

### 📄 m4_consultas_negocio.sql

Script que contiene consultas para obtener indicadores comerciales a partir de la tabla `ventas`.

Las consultas desarrolladas incluyen:

- Resumen ejecutivo mensual.
- Ranking de productos por facturación.
- Identificación de clientes recurrentes.
- Comparación de la facturación mensual respecto al promedio del período.
- Hallazgos obtenidos a partir del análisis de los resultados.

---

## Conceptos aplicados

En este ejercicio se utilizaron los siguientes conceptos:

- `SELECT`
- `SUM()`
- `COUNT()`
- `AVG()`
- `GROUP BY`
- `HAVING`
- `ORDER BY`
- `TOP`
- `CASE`
- `WITH` (Common Table Expression - CTE)
- Alias de columnas.
- Funciones de agregación para análisis de negocio.

---

## Objetivo

El objetivo de esta práctica es transformar datos operativos en información útil para la toma de decisiones mediante consultas SQL, preparando la base de datos para los próximos módulos del curso, donde se incorporarán cruces de información (`JOIN`) y posteriormente la integración con Power BI.

---

Desarrollado utilizando **Microsoft SQL Server** y **SQL Server Management Studio (SSMS)** como parte del curso de **Data Analytics de Coderhouse**.
